# AI-Powered Micro-Learning Platform 📱🤖

## 🚀 Overview
This project is an AI-powered mobile application designed to provide personalized micro-learning experiences for students, especially those with limited time or access to continuous education.

The platform delivers short-duration (5–10 minutes) learning modules and adapts content based on user performance using machine learning.

---

## 🎯 Key Features

### 📚 Micro-Learning Modules
- Short lessons optimized for quick learning
- Designed for working students and time-constrained users

### 🧠 AI-Based Recommendation System
- Suggests personalized lessons based on performance
- Continuously adapts learning path

### 📊 Aptitude Assessment
- Evaluates user's current knowledge level
- Helps in customizing learning experience

### 📈 Progress Tracking & Analytics
- Tracks performance over time
- Provides insights into improvement

### 📱 Mobile Application (Flutter)
- Clean and responsive UI
- Optimized for Android devices

### 🔄 Offline Support
- Access content without internet
- Sync data when connection is available

### 🛠 Admin Dashboard
- Manage users, courses, and analytics
